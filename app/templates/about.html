About

What's a timer board?

A timer board is a pretty old concept in EVE, it's a webpage used to track reinforcement timers, these are commonly tracked for things including:

  • Player Owned Starbases
  • Infrastructure Hub
  • Customs Offices

Who runs this website?

Hi, I'm Lucia Denniard, head diplomat for Pasta Syndicate, I maintain multiple EVE related projects including wormhol.es-zkb, pizza-auth and timerboard (which this website was born from).

Only I have access to this server, and it shares the same hardware as my other websites, if you don't believe you can trust me, I would recommend running your own board.

Why do I need to sign in with an EVE Online account? Are you hacking my account?

This website is built using CCP's Single Sign On, read about it in this devblog. It allows me to authenticate who you are without asking for API keys or trusting the easily-faked ingame web browser. When you click the Login button it should redirect you to https://login.eveonline.com/ on a secure connection, you can click the green padlock in your web browser to check that it is a valid part of the official eveonline.com website.
